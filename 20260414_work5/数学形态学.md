### **课件大纲 - 数学形态学**

#### **一、数学形态学概述**（1课时）
1. **定义与起源**  
   - 数学形态学：基于集合论、拓扑理论的图像分析方法，用于分析形状与结构  
   - 创始人：Georges Matheron, Jean Serra（1960s，应用于地质学和材料科学）  

2. **核心思想**  
   - **结构元素（Structuring Element）**：探针，用于探测图像中的特定形状  
   - **基本操作**：腐蚀、膨胀、开运算、闭运算  

3. **应用领域**  
   - 医学图像处理（细胞分割、血管提取）  
   - 工业检测（缺陷检测、目标定位）  
   - 文档处理（文字增强、噪声去除）  

---

#### **二、二值形态学基础**（2课时）  
1. **结构元素设计**  
   - 形状：矩形、圆形、十字形  
   - 尺寸：3×3、5×5（通过动画演示不同结构元素的影响）  

2. **基本操作**  
   - **腐蚀（Erosion）**：  
     - 公式：\( A \ominus B = \{ z | B_z \subseteq A \} \)  
     - 作用：消除边界点，缩小目标区域  
   - **膨胀（Dilation）**：  
     - 公式：\( A \oplus B = \{ z | (\hat{B})_z \cap A \neq \emptyset \} \)  
     - 作用：填补空洞，扩大目标区域  

3. **组合操作**  
   - **开运算（Opening）**：先腐蚀后膨胀（去噪、平滑轮廓）  
   - **闭运算（Closing）**：先膨胀后腐蚀（填充孔洞、连接断裂）  

4. **几何意义图示**  
   - 腐蚀/膨胀动态示意图  
   - 开/闭运算效果对比（如去除小物体 vs 填充小孔洞）  

---

#### **三、灰度形态学**（1课时）  
1. **灰度腐蚀与膨胀**  
   - 操作定义：对图像局部区域取最小值（腐蚀）或最大值（膨胀）  
   - 应用：去除亮/暗噪声、增强对比度  

2. **形态学梯度**  
   - 公式：梯度 = 膨胀图像 - 腐蚀图像  
   - 作用：边缘检测（保留物体边界）  

3. **顶帽与底帽变换**  
   - 顶帽：原图 - 开运算（增强亮细节）  
   - 底帽：闭运算 - 原图（增强暗细节）  

---

1. **案例1：去除二值图像中的小噪声**（开运算）

### **实验代码设计**  
#### **实验1：二值形态学基础操作**  
```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# 生成测试二值图像（含噪声）
image = np.zeros((200, 200), dtype=np.uint8)
image[50:150, 50:150] = 255  # 白色正方形
noise = np.random.randint(0, 255, (200, 200))  # 添加噪声
image[noise > 240] = 255
image[noise < 15] = 0

# 定义结构元素（3x3十字形）
kernel = cv2.getStructuringElement(cv2.MORPH_CROSS, (3,3))

# 腐蚀与膨胀
eroded = cv2.erode(image, kernel, iterations=1)
dilated = cv2.dilate(image, kernel, iterations=1)

# 开运算与闭运算
opened = cv2.morphologyEx(image, cv2.MORPH_OPEN, kernel)
closed = cv2.morphologyEx(image, cv2.MORPH_CLOSE, kernel)

# 可视化
images = [image, eroded, dilated, opened, closed]
titles = ['Original', 'Erosion', 'Dilation', 'Opening', 'Closing']
plt.figure(figsize=(15,5))
for i in range(5):
    plt.subplot(1,5,i+1), plt.imshow(images[i], 'gray'), plt.title(titles[i])
plt.show()
```

```python
# 读取灰度图像
img = cv2.imread('cell.jpg', 0)

# 灰度形态学操作
kernel = cv2.getStructuringElement(cv2.MORPH_RECT, (5,5))
gradient = cv2.morphologyEx(img, cv2.MORPH_GRADIENT, kernel)

# 对比传统边缘检测
edges_sobel = cv2.Sobel(img, cv2.CV_64F, 1, 1, ksize=3)

# 可视化
plt.figure(figsize=(12,4))
plt.subplot(131), plt.imshow(img, 'gray'), plt.title('Original')
plt.subplot(132), plt.imshow(gradient, 'gray'), plt.title('Morphological Gradient')
plt.subplot(133), plt.imshow(edges_sobel, 'gray'), plt.title('Sobel Edge')
plt.show()
```

#### **实验3：孔洞填充（实际应用）**  
```python
def hole_filling(image):
    # 二值化（假设输入为灰度图）
    _, binary = cv2.threshold(image, 127, 255, cv2.THRESH_BINARY_INV)
    
    # 初始化掩模
    mask = np.zeros_like(binary)
    mask[100:120, 100:120] = 255  # 在孔洞内设置种子点
    
    # 形态学重建
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5,5))
    while True:
        new_mask = cv2.dilate(mask, kernel)
        new_mask = cv2.bitwise_and(new_mask, binary)
        if np.array_equal(new_mask, mask):
            break
        mask = new_mask
    
    # 合并结果
    filled = cv2.bitwise_not(cv2.bitwise_or(binary, mask))
    return filled

# 测试
filled_img = hole_filling(img)
plt.imshow(filled_img, 'gray'), plt.title('Hole Filled')
```
