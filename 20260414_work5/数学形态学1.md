**新增内容说明：**

1. **低帽变换原理**：
- 数学定义：Bottom Hat = Closing - Original
- 功能特性：突出显示比结构元素小的暗区域（如闭运算后产生的特征）
- 典型应用：检测图像中的暗色细节（如裂缝、孔洞等）

2. **代码实现**：
```python
bottomhat = cv2.morphologyEx(binary_img, cv2.MORPH_BLACKHAT, kernel)
```
- 使用OpenCV内置的`MORPH_BLACKHAT`模式直接实现
- 与高帽变换形成对偶操作：高帽强调亮细节，低帽强调暗细节

**各形态学操作对比说明表**：

| 操作名称          | 数学表达式       | 功能特点               | 典型应用场景       |
| ----------------- | ---------------- | ---------------------- | ------------------ |
| 膨胀 (Dilation)   | A⊕B              | 扩大亮区域，填充小孔   | 连接断裂区域       |
| 腐蚀 (Erosion)    | A⊖B              | 缩小亮区域，消除小物体 | 去除噪点           |
| 开运算 (Opening)  | A∘B = (A⊖B)⊕B    | 平滑轮廓，消除小亮区域 | 去除细小连接部分   |
| 闭运算 (Closing)  | A•B = (A⊕B)⊖B    | 填充孔洞，平滑轮廓     | 闭合物体内部小孔   |
| 高帽 (Top Hat)    | T(A) = A - (A∘B) | 增强亮细节             | 提取细线、亮斑     |
| 低帽 (Bottom Hat) | B(A) = (A•B) - A | 增强暗细节             | 检测裂缝、暗斑     |
| 击中击不中        | A⊗(B,C)          | 匹配特定模式           | 特征检测、形状识别 |
